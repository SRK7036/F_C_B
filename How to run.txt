How to run

Backend

Create DB (Supabase Postgres) and get the connection string.

cd backend

python -m venv .venv && source .venv/bin/activate (Windows: .venv\Scripts\activate)

pip install -r requirements.txt

cp .env.example .env and fill in:

SUPABASE_DB_URL

ANTHROPIC_API_KEY or OPENAI_API_KEY

Add your PDFs/DOCX/CSV/TXT to knowledge_base/ (sibling of backend).

python -m rag.ingestion (builds Chroma)

uvicorn main:app --reload --port 8000

Frontend

cd frontend

cp .env.local.example .env.local (points at http://localhost:8000)

npm install

npm run dev ‚Üí open http://localhost:3000
